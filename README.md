# Kristian Todman — IT Support & Defensive Security Portfolio

**IT Support Specialist | Help Desk Technician | Junior SOC Analyst**

A hands-on technical portfolio built through real homelab work. Every project in this repository was designed, configured, documented, and tested in a self-built lab environment running on Proxmox VE. This portfolio demonstrates practical skills across IT support, endpoint management, Windows/Linux administration, network troubleshooting, and defensive security operations.

> **Status:** Private build in progress. Projects are published progressively as they are completed and verified. See [PROJECT_STATUS.md](PROJECT_STATUS.md) for current progress.
