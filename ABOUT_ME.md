## What I Actually Know

I'm not going to list every buzzword. Here's what I can honestly back up with lab work:

**Windows:** I've set up Windows Server 2022, deployed Active Directory, created OUs and user accounts, configured Group Policy, and walked through common troubleshooting scenarios (boot issues, permission errors, profile problems, networking failures). I'm comfortable in PowerShell for basic support tasks.

**Linux:** I work in Ubuntu Server and Debian regularly. I know my way around the command line — user management, file permissions, service management with systemd, log review, and basic shell scripting.

**Networking:** I understand TCP/IP, DNS, DHCP, subnetting, and VLANs at a functional level. I've configured network segmentation in my homelab and troubleshot connectivity issues across VMs. I can read a Wireshark capture for basic triage.

**Security monitoring:** I've deployed Wazuh as a SIEM, connected agents, and worked through real alert investigations. I know how to read Windows Event Logs and Linux auth logs, identify anomalies, and document findings in a format that an analyst or supervisor could act on.

**Documentation:** I document everything. Troubleshooting logs, change notes, incident summaries, lab writeups. If I can't explain what I did and why, I don't consider it done.
