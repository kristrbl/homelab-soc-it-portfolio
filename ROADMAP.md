# Portfolio Roadmap

The goal is to have 6 complete, documented, screenshot-backed labs ready to show before applying. Projects are prioritized by hiring impact — the ones that come up most in IT support and entry-level SOC interviews come first.

---

## Stage 1 — Foundation (Complete These First)

These three labs directly answer the most common interview questions for both tracks.

| Priority | Project | Why It Matters |
|---|---|---|
| 🔴 1 | Active Directory Home Lab | Asked in almost every IT support and junior sysadmin interview. "Have you worked with AD?" |
| 🔴 2 | Windows Troubleshooting Lab | Core skill for any help desk role. Documents your diagnostic process. |
| 🔴 3 | Wazuh SIEM Setup & Alert Investigation | Shows you've actually deployed a SIEM and worked through alerts. Not just theory. |

**Exit criteria:** Each project has a complete README, at least 5 screenshots, and you can walk through it in a 10-minute interview without notes.

---

## Stage 2 — Depth (Strengthens Both Tracks)

| Priority | Project | Why It Matters |
|---|---|---|
| 🟠 4 | User Account Provisioning Workflow | HR/helpdesk process. New hire onboarding, offboarding, access requests. |
| 🟠 5 | Brute Force Detection Scenario | Real SOC scenario. Shows you can identify an attack in logs and write it up. |
| 🟠 6 | Linux Auth Log Review | Basic but important. Many SOC analysts live in logs. Shows you can read and interpret them. |

## Stage 4 — Polish & Publish

- Review every README for clarity and accuracy
- Redact any sensitive info (home IPs, credentials, personal data)
- Replace placeholder screenshots with real evidence
- Remove any "in progress" markers from completed labs
- Make repo public
- Pin best 6 projects on GitHub profile

---

## Stage 5 — Certifications

Study and exam schedule built around available time. A+ first because it's most relevant for IT support applications. Security+ after A+ to support the SOC track.

| Cert | Relevance |
|---|---|
| CompTIA A+ | IT Support, Help Desk — almost required for entry-level |
| CompTIA Network+ | Networking, IT Support, SOC — strengthens both tracks |
| CompTIA Security+ | SOC, Defensive Security — standard entry requirement for security roles |
